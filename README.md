# viptransfer.com

Modern, premium VIP transfer landing page built with React and Vite.

## Site architecture

- Global sticky-ready header and persistent WhatsApp CTA
- Hero with primary value proposition and trust signals
- Quick booking form
- Service catalogue
- Premium vehicle fleet
- Four-step booking process
- Company trust and credentials
- Accessible FAQ accordion
- Contact CTA and enquiry form
- Multi-column footer

## Wireframe

```text
[ Header: logo | navigation | phone | Book Now ]
[ Cinematic hero copy                 | vehicle image ]
[ Primary CTA | Fleet CTA | social proof              ]
[ Quick reservation form                             ]
[ Services: 4 x 2 card grid                          ]
[ Fleet: responsive vehicle card grid                ]
[ How it works: 4 steps                              ]
[ Trust image | experience, credentials and benefits ]
[ FAQ accordion                                      ]
[ Premium contact CTA | enquiry form                 ]
[ Footer columns                                     ]
[ Floating WhatsApp CTA                              ]
```

## Design system

- Display: Cormorant Garamond
- UI/body: Manrope
- Base surfaces: `#050505`, `#0B0B0B`, `#111111`
- Accent: `#D4AF37`
- Secondary text: `#B8B8B8`
- Visual language: restrained gold rules, editorial typography, thin borders,
  subtle glow, cinematic imagery and generous whitespace.
